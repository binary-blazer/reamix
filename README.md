## ショートカットキー

| ショートカット               | キー一覧               |
| :--------------------------- | :--------------------- |
| Reamix について              | Ctrl(Cmd) + Alt + A    |
| 隠す                         | Ctrl(Cmd) + H?         |
| 他を隠す                     | Ctrl(Cmd) + Shift + H? |
| 再起動?                      | Ctrl(Cmd) + Alt + R    |
| 終了                         | Ctrl(Cmd) + Q          |
| 再読み込み                   | Ctrl(Dmd) + R          |
| 戻る                         | Ctrl(Cmd) + Shift + Z  |
| 進む                         | Ctrl(Cmd) + Shift + X  |
| Web サイトの開発者向けツール | F12                    |
| Reamix の開発者向けツール    | Alt + F12              |
| 設定ウィンドウ               | Ctrl(Cmd) + Alt + S    |

タブの移動には対応していませんが、正式版では対応する見通しです。
